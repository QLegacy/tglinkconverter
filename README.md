# ⚡️ Telegram Link Resurrector (Конвертер юзернеймов)

Простой, легковесный и быстрый HTML/JS инструмент, который «воскрешает» ссылки с мертвого, заблокированного или некорректно работающего `t.me` (а также парсит кривые ссылки, юзернеймы с `@` и прочий мусор), превращая их в рабочие редиректы через `telegram.me`.

## 🚀 Как это работает?

У скрипта два режима работы:

1. **Автоматический редирект (через URL-параметр):**
   Если вы переходите по ссылке вида `qlegacy.github.io/tglinkconverter/?username=qlegacy`, скрипт моментально перенаправит вас на рабочий `https://telegram.me/qlegacy`.
   
2. **Интерактивный интерфейс:**
   Если открыть страницу без параметров, вас встретит красивое меню в стиле Glassmorphic. Туда можно вставить вообще всё что угодно:
   * `t.me/quadlegacy`
   * `https://telegram.me/quadlegacy`
   * `@quadlegacy`
   * `quadlegacy`

   Умный парсер на лету отсечет лишние протоколы, слэши, собачки (`@`), параметры запросов и выдаст чистый рабочий редирект.

## 🛠 Установка и запуск

Проект состоит всего из одного файла и не требует серверов, баз данных или Node.js. 
1. Зайдите на qlegacy.github.io/tglinkconverter/

   ИЛИ

1. Склонируйте репозиторий или просто скачайте файл `index.html`.
2. Залейте его на любой бесплатный хостинг (GitHub Pages, Vercel, Netlify).
3. Готово!
